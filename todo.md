# ConferenceTracker - タスク管理

**プロジェクト名**: ConferenceTracker（企業学会発表トラッカー）  
**目標**: 博士就活生向けSaaS - 月額500円で20人の顧客から1万円の売上達成  
**期間**: 2025年11月9日〜12月6日（4週間）  
**現在のフェーズ**: Week 1 - データベース移行とキーワード抽出

---

## プロジェクト概要

### ビジネス目標
- **1ヶ月後（12/9）**: 20人 × 500円 = 10,000円
- **3ヶ月後（2/9）**: 100人 × 1,000円 = 100,000円
- **6ヶ月後（5/9）**: 1,000人 × 2,000円 = 2,000,000円

### ユーザー価値提案
- 研究テーマから企業を発見
- マッチ度スコアで企業をランキング表示
- 企業の発表タイトルを閲覧
- お気に入り企業を保存

### 技術スタック
- フロントエンド: React 19 + Tailwind CSS 4 + shadcn/ui
- バックエンド: Node.js + tRPC
- データベース: PostgreSQL + Drizzle ORM
- 決済: Stripe
- ホスティング: Manus
